<h2>解析：</h2>
绘制由叶子组成的图案。
<h2>答案：</h2>
<pre class="prettyprint linenums">import turtle as t
def yuanhu(r, n):
    # 用 循环绘制
    # for i in range(0,n):
    #     t.fd(0.0174*r)
    #     t.rt(1)
    # 用 circle 方法绘制
    t.circle(-r, n)
 
def yezi(r, n):
    for i in range(0,2):
        yuanhu(r,n)
        t.rt(180-n)

# 图案一
n=3
# 图案二
# n=4
# 图案三
# n=5
for i in range(0, n):
    yezi(80, 90)
    t.rt(360/n)
