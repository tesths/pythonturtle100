<h2>解析：</h2>
绘制由半圆组成的梅花。
<h2>答案：</h2>
方法一
<pre class="prettyprint linenums">import turtle as t
def meihua(n, d):
    for i in range(0, n):
        for j in range(0, 180):
             t.fd(0.0174*d)
             t.rt(1)
        t.lt(180-360/n)
    return

t.lt(90)
meihua(3, 40)
# meihua(4, 40)
# meihua(5, 40)</pre>
方法二
<pre class="prettyprint linenums">import turtle as t
def meihua(n, d):
    for i in range(0, n):
        t.circle(-d, 180)
        t.lt(180-360/n)
    return
t.lt(90)
meihua(3, 40)
# meihua(4, 40)
# meihua(5, 40)</pre>
